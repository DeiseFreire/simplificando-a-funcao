### Funcionalidade do Código

O código LaTeX apresentado implementa a simplificação de uma fração envolvendo as funções `f(x)` e `g(x)`. As funções são definidas da seguinte maneira:

* `f(x) = x^2 - 3`
* `g(x) = 2x - 1`
